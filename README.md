# Ship / Boat Detection with YOLOv8

Object detection project for a Deep Learning course.  
Dataset: ~694 boat images with Pascal VOC bounding-box annotations.  
Model: YOLOv8 (Ultralytics).

---

## Project Structure

```
Ship-Detection-DL/
├── data2/
│   ├── images/          # Raw PNG images (boat0.png … boat693.png)
│   └── annotations/     # Pascal VOC XML files (boat0.xml … boat693.xml)
│
├── dataset/             # Auto-generated by prepare_dataset.py
│   ├── train/images & labels/
│   ├── val/images   & labels/
│   └── test/images  & labels/
│
├── runs/                # Auto-generated training / eval / predict outputs
│
├── prepare_dataset.py   # VOC → YOLO conversion + train/val/test split
├── train.py             # YOLOv8 training
├── evaluate.py          # Test-set evaluation & metrics
├── predict.py           # Inference on new images
├── visualize.py         # Dataset visualization
├── dataset.yaml         # Auto-generated YOLO dataset config
└── requirements.txt
```

## Setup

```bash
pip install -r requirements.txt
```

## Step-by-Step Usage

### 1 — Prepare the dataset
Converts XML annotations to YOLO format and splits 70 / 20 / 10.

```bash
python prepare_dataset.py
```

Output:
```
Total valid samples: 694
Train: 485 | Val: 138 | Test: 71
Dataset YAML written to: dataset.yaml
```

---

### 2 — (Optional) Visualize samples

```bash
python visualize.py --split train --n 9
```

Saves `visualize_train.png` with bounding boxes drawn.

---

### 3 — Train

```bash
# Default: YOLOv8n, 100 epochs, 640px, batch 16
python train.py

# Larger model for better accuracy
python train.py --model yolov8s.pt --epochs 150 --batch 8
```

Best weights are saved to `runs/train/boat_yolov8/weights/best.pt`.

---

### 4 — Evaluate on test set

```bash
python evaluate.py --weights runs/train/boat_yolov8/weights/best.pt
```

Prints:
```
── Test Set Metrics ──────────────────────────────
  mAP@0.5:       0.xxxx
  mAP@0.5:0.95:  0.xxxx
  Precision:     0.xxxx
  Recall:        0.xxxx
──────────────────────────────────────────────────
```

---

### 5 — Predict on new images

```bash
# Single image
python predict.py --source data2/images/boat0.png --save

# Entire test split
python predict.py --source dataset/test/images/ --save
```

Annotated images are saved to `runs/predict/boat_detections/`.

## Model Options

| Model      | Size   | Speed | Accuracy |
|------------|--------|-------|----------|
| yolov8n.pt | ~6 MB  | Fast  | Good     |
| yolov8s.pt | ~22 MB | Med   | Better   |
| yolov8m.pt | ~52 MB | Slow  | Best     |

For a course project with ~694 images, `yolov8n.pt` or `yolov8s.pt` is recommended.

---

## Key Metrics Explained

- **mAP@0.5** — mean Average Precision at IoU threshold 0.5 (primary metric)
- **mAP@0.5:0.95** — averaged over IoU thresholds 0.5 to 0.95
- **Precision** — of all predicted boxes, how many are correct
- **Recall** — of all ground-truth boxes, how many were found
